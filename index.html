Grilla de elementos hexagonales, usando CSS3

Basado en el tema "Chronos", se hacen modificaciones para poder usar Sass, obteniendo un resultado más ajustado a nuestras necesidades.

El problema que encontre con varios scripts css3, era de incompatibilidad con Bootstrap 3. En este caso hice leves modificaciones para poder usarlo sin problemas con esta librería.

$hex-color-background: #6A6A62;
$hex-color-background-hover: #B6BBB5;
$hex-color-title: #6A6A62;
$hex-grid-container-width: 960px; 
$hex-grid-row-items: 4; 
$hex-grid-margin: 5px; 

$hex-width: ($hex-grid-container-width - ( ($hex-grid-margin * 2) * $hex-grid-row-items ) ) / $hex-grid-row-items;
$hex-height: $hex-width * 2;
